## Caption Format Reference

```
Label: description of content. Additional note.

Examples:
  Public A: reception and dispatch. Workstations, filing storage, and a private office alcove.
  Site plan: halfway house cluster (North), standard housing (center), and artist foundation (South).
  Floor plans: bathhouse and water filtration research lab.
  Facade elevation: Red Hook Grain Terminal with planted diamond grid.
```

**Rules:**
- No hyphens anywhere
- No em dashes (use colon or new sentence)
- Capitalize only the first word of the label and proper nouns

## Reading / Essay Page Template

*(For text/ pages — different structure from project pages)*

```
Series label:   On Cinema — Deleuze   (omit if standalone)
Title (h2):     I. Movement and Duration
Reading meta:   Reading notes, Fall 2021
```

**Paragraph structure for reading notes:**
- 2–4 paragraphs per reading
- Preserve the author's original voice — grammar corrections only
- Block-quote direct citations: Author, page number
- No hyphens in any text (including compound adjectives in the original)

**Navigation (series only):**
```
← II. Previous Title     IV. Next Title →
```
- First entry: no left arrow
- Last entry: no right arrow
